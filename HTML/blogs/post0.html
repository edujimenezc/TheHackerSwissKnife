Primeros Pasos

Necesitaremos:

  • -Una Raspberry Pi
  • -Tener python instalado
  • -Descargar e instalar la sdk de vector
  • -El programa Spanish Vector

Primero de todo encenderemos nuestra Raspberry Pi con Raspberry Pi OS instalado y la actualizaremos, para ello abrir el terminal y escribir: 

sudo apt-get update


Después instalaremos python si no lo tenemos instalado: 
Enlace


Finalmente instalaremos la sdk y seguiremos los pasos que indica para configuar nuestro Vector y así tenerlo conectado a nuestra Raspberry para comenzar a hacer pruebas 
Instalar sdk


Para terminar, probaremos el programa SpanishVector.py y podremos empezar a hacer uso de é, aunque aún queda la mayor parte.
